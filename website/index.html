A symbolic computer algebra system by Allen Franklin Jordan


Franklin Math is a computer algebra system (CAS) written in Java that supports both numeric and symbolic arithmatic and other computations.

Example Input:

RedundantFunction[x] = Sin[x]^2 + Cos[x]^2
RedundantFunction[Pi[]]
